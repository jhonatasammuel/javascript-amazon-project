# 🛒 E-commerce Intelligence Lab (Amazon-Style)

Este projeto representa um marco importante no meu desenvolvimento como programador. O foco principal não foi a construção da interface (utilizando HTML/CSS base de curso), mas sim a implementação da **camada de inteligência e manipulação de dados**.

### 🚀 Evolução Técnica e Implementações
Neste projeto, saí da lógica básica para implementar fluxos reais de uma aplicação web:

- **Manipulação de Dados Complexos**: Uso intensivo de **JSON** e listas de objetos (dicionários) para renderizar produtos e informações de forma dinâmica.
- **Modularização (Clean Code)**: Organização do código através da **Criação e Exportação de Funções**, garantindo um código mais limpo e reutilizável.
- **Persistência de Dados**: Implementação de **LocalStorage** para que o carrinho e as escolhas do usuário não se percam ao atualizar a página.
- **Lógica de Checkout**: Desenvolvimento de funções para cálculo de totais, impostos e prazos de entrega baseados na interação do usuário.

### 🛠️ Tecnologias e Conceitos Aplicados
- **JavaScript ES6+**: Arrow functions, módulos (Import/Export), e métodos de array.
- **Gestão de Estado Simples**: Controle manual de quantidades e remoção de itens.
- **Integração Front-End**: Conectar a lógica JS ao DOM para refletir mudanças em tempo real.

### ⚙️ Status do Projeto: "Work in Progress"
Este repositório é o meu laboratório de estudos atual. Por ser um projeto em evolução, alguns pontos de sincronização de interface estão sendo ajustados para melhorar a reatividade do site.
